# Quick Start Guide

## How to Play

1. **Open the game**:
   - Simply double-click `index.html` in your file browser
   - OR open it in your browser: File → Open → select `index.html`

2. **Start playing**:
   - Click "Start Game" button
   - The board will be populated with colored blocks

3. **Gameplay**:
   - **Select a block**: Click/tap any block in the playable area (rows above the gray preview row)
   - **Move it**: Click arrow indicators OR use keyboard arrow keys (Left/Right)
   - **Drop it**: Click the block again OR press Enter/Space
   - **Clear lines**: Fill a complete horizontal row to clear it and score points!

4. **Scoring**:
   - 1 line cleared: 100 points
   - 2 lines cleared: 300 points (1x + 2x combo!)
   - 3 lines cleared: 700 points (1x + 2x + 4x combo!)
   - 4 lines cleared: 1500 points (1x + 2x + 4x + 8x combo!)

## Tips

- Plan ahead! Look at the preview row (gray area at bottom) to see what blocks are coming next
- Try to set up multiple line clears for big combo scores
- Blocks will automatically raise from the preview row after each move
- Don't let the board fill up - game over when blocks can't raise!

## Controls Summary

**Mouse/Touch**:
- Click block to select
- Click again to drop
- Click left/right arrows to move

**Keyboard**:
- Arrow Left/Right or A/D: Move selected block
- Enter or Space: Drop block
- Escape: Deselect block
- P: Pause (when playing)

## Troubleshooting

If the game doesn't load:
1. Make sure you're using a modern browser (Chrome, Firefox, Safari, Edge)
2. Enable JavaScript in your browser
3. Check the browser console (F12) for any error messages

Enjoy!
